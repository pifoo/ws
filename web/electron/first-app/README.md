1. 全局安装electron
```
// electron
npm install -g electron-prebuilt
// Packaging Tools  
npm install -g electron-packager  
npm install -g electron-builder
```

2. 本地安装
```
npm install electron-prebuilt --save-dev
```

3. 编写代码   
工程结构   
```
appName
├── README.md
├── index.html
├── main.js
├── node_modules
└── package.json
```

4. run
```
npm run start
```
